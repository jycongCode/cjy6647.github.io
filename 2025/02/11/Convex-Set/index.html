2025/2/11 Convex Optimization
Affine Sets

  • A set $C \subseteq \mathbf{R}^nC⊆RnC \subseteq \mathbf{R}^nC⊆Rn$ is affine if the line through any two distinct points in $CCCC$ lies in $CCCC$

  • If for any $x_1,x_2 \in C$ and $\theta \in \mathbf{R}$ ,we have $\theta x_1+(1- \theta )x_2 \in C$

  • subspace If $C$ is an affine set and $x_0 \in C$, then the set

$$
V = C-x_0 = {x-x_0|x \in C }
$$

  • affine hull : smallest affine set that contains $C$

    $$
    \mathbf{aff} \space C={ \theta_1x_1+…+\theta_kx_k | x_1,…,x_k \in C, \theta_1 + … + \theta_k = 1 }
    $$

  • affine dimension the dimension of a set $C$’s affine hull

  • relative interior

    $$
    \mathbf{relint} \space C = { x \in C | B(x,r) \cap \mathbf{aff} \space C \subseteq C for \space some \space r > 0 }
    $$

    where $B(x,r) = {y| \parallel y-x \parallel \leq r }$

Convex Set

  • A set $C$ is convex if the line segment between any two points in $C$ lies in $C$

  • if for any $x_1,x_2 \in C$ and any $\theta$ with $0 \leq \theta \leq 1$, we have

    $$
    \theta x_1 + (1- \theta ) x_2 \in C
    $$

  • convex hull : smallest convex set that contains $C$

    $$
    \mathbf{conv} \space C = { \theta_1 x_1 + … + \theta_k x_k | x_i \in C, i=1,…,k,\theta_1 + … + \theta_k = 1 }
    $$

Cone

  • A set $C$ is called a cone , or nonnegative homogeneous , if for every $x \in C$ and $\theta \geq 0$ we have $\theta x \in C$

  • convex cone (convex and cone) for any $x_1,x_2 \in C$ and $\theta_1,\theta_2 \geq 0$, we have

    $$
    \theta_1 x_1 + \theta_2 x_2 \in C
    $$

  • geometrcic 2D pie slice with apex 0 and edges passing through $x_1$ and $x_2$

  • conic combination a point of form $\theta_1 x_1 + … + \theta_k x_k$ with $\theta_1, … \theta_k \geq 0$

  • if $x_i$ are in a convex cone $C$ , then every conic combination of $x_i$ is in $C$

  • a set $C$ is a convex cone iff it contains all conic combinations of its elements.

  • conic hull : smallest convex cone that contains $C$

    $$
    { \theta_1 x_1 + … + \theta_k x_k | x_i \in C, \theta_i \geq 0, i = 1,…k }
    $$

Important examples

  • The empty set $\emptyset$ , any single point ${ x_0 }$ , and the whole space $\mathbf{R}^n$ are affine (hence, convex) subsets of $\mathbf{R}^n$

  • Any line is affine. If it passes through zero, it is a subspace, hence also a convex cone.

  • A line segment is convex, but not affine (unless it resuces to a point).

  • A ray, which has the form ${x_0 + \theta v | \theta \geq 0$, where $v \neq 0$ is convex, but not affine. It is a convex cone if its base $x_0$ is 0.

  • Any subspace is affine, and a convex cone (hence convex).

Hyperplanes

  • a set of form ${x|a^Tx=b}$ , where $a \in \mathbf{R}^n$ , and $b \in \mathbf{R}$

  • Geometrically interpreted: the set of points with a constant inner product to a give vector a, or as a hyperplane with normal vector a; the constant $b \in \mathbf{R}$ determines the offset of the hyperplane from the origin. Thus expressed

    $$
    { x | a^T (x-x_0) = 0 }
    $$

  • halfspace A hyperplane divides $\mathbf{R}^n$ into two halfspaces. A (closed) halfspace is a set of the form

    $$
    {x|a^Tx \leq b }
    $$

    where $a \neq 0$. Halfspaces are convex but not affine.

Polyhedra

  • the solution set of a finite number of linear equalities and inequalities

    $$
    \mathcal{P} = { x | a^T_j x \leq b_j, j = 1, … , m, c^T_j x = d_j,j = 1, … , p }
    $$

    intersection of a finite number of halfspaces and hyperplanes.

  • Affine sets(e.g. subspaces,hyperplanes,lines), rays, line segments, and halfspaces are all polyhedra.

  • Polyhdra are convex sets.

positive semidefinite cone

  • We use $S^n$ denote the set of symmetric $n \times n$ matrices.

  • $S^n_+$ denotes the set of symmetric positive semidefinite matrices

  • $S^n_{++}$ denote the set of symmetric positive definite matrices

  • $S^n_+$ is a convex cone

Separating and supporting hyperplanes

  • seperating hyperplane theorem

    Suppose C and D are nonempty disjoint convex sets, then there exist $a \neq 0$ and $b$ such that $a^T x \leq b$ for all $ x \in C$ and $a^Tx \geq b$ for all $x \in D$

  • converse separating hyperplane theorems

    The converse of the separating hyperplane theorem(i.e. existence of a separating hyperplane implies that C and D do not intersect) is not true.

  • supporting hyperplanes

    Suppose $C \subseteq \mathbf{R}^n$, and $x_0$ is a point in its boundary $bd$ C, i.e.,

    $$
    x_0 \in \mathbf{bd} \space C = \mathbf{cl} \space C \backslash \mathbf{int} \space C
    $$

  • supporting hyperplane theorem

    For any nonempty convex set C, and any $x_0 \in \mathbf{bd} \space C$, there exists a supporting hyperplane to C at $x_0$

Dual cones

  • Let K be a cone. The set

    $$
    K^* = { y | x^Ty\geq 0 \space \mathrm{for} \space \mathrm{all} \space x \in K }
    $$

    is called the dual cone of K

  • $K^*$ is a cone, and is always convex, even when the original cone K is not.

  • Dual generalized inequalities

    • $x \preceq_k y$ iff $\lambda^T \leq \lambda^T y$ for all $\lambda \succeq_{K^*} 0$

    • $x \prec_k y$ iff $\lambda^T < \lambda^T y$ for all $\lambda \succeq_{K^*} 0,\lambda \neq 0$
